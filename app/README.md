# anDiary

Write your daily journal

## Usage

> We recommend using pnpm because the app has been tested using pnpm.

### Install Dependencies

```shell
pnpm install --shamefully-hoist
```

### Use it

**Development Mode:**

```shell
$ pnpm run dev
```

**Production Build:**

```shell
$ pnpm run build
```
